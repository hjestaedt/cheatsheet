# ibmcloud

###### __help__
```bash
ibmcloud help
```

###### __update cli__
```bash
ibmcloud update
```

###### __install plugin__
```bash
ibmcloud plugin install <plugin-name>
```

###### __retrieve bearer token__
```bash
ibmcloud login
ibmcloud iam oauth-tokens
```
